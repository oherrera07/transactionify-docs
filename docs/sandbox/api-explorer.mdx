# API Explorer

Explore, test, and inspect all Transactionify API endpoints directly from this page.
Paste your API key into the bar below, select an endpoint, fill in parameters, and hit **Send Request** to see the live response.

:::info Authentication
All endpoints require an API Key in the `Authorization` header:
```
Authorization: APIKey <your-api-key>
```
:::
